# 📊 Customer Churn Analysis – Power BI Project

## 📌 Project Overview

This project focuses on analyzing **customer churn behavior** using **Power BI** to help businesses understand why customers leave and how retention can be improved. The dashboard provides clear insights into churn patterns based on customer demographics, services, and billing details.

---

## 🎯 Objectives

* Identify key factors contributing to customer churn
* Analyze churn trends across different customer segments
* Support data-driven decisions to improve customer retention

---

## 🛠 Tools & Technologies

* **Power BI**
* **Microsoft Excel / CSV**
* **Data Modeling & DAX**
* **Data Cleaning & Transformation**

---

## 📂 Dataset

* **Customer-Churn.csv**
* Contains customer demographics, service usage, contract type, payment methods, and churn status.

---

## 📈 Key Insights & Features

* Churn rate comparison by **contract type**
* Impact of **monthly charges** on churn
* Churn analysis based on **tenure**
* Customer segmentation by **gender, senior citizen, and services**
* Interactive filters and slicers for dynamic analysis

---

## 📊 Dashboard Highlights

* KPI cards for total customers & churn rate
* Visual breakdown of churn vs non-churn customers
* Trend analysis to identify high-risk customer groups

---

## 🚀 Outcomes

* Identified high churn among **month-to-month contract customers**
* Higher churn observed in customers with **short tenure**
* Monthly charges strongly influence churn probability
